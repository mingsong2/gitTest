### Git命令

git add README.md                                                       //工作区->暂存区
git commit -m "first commit"                                            //暂存区->提交至本地分支
git remote add origin https://github.com/songming43/gitTest.git         //添加远程仓库
git push -u origin master                                               //推送至远程分支 

git pull origin master

git reset
